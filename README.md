# Kshell
A feature rich shell built using C. Built on the Codecrafters framework with custom enhancements including pipe support and input/output redirection.

# Features
**Basic Commands** like ls, pwd, cd, etc.<br>
**Input Output Redirection** using >, >>, <, << <br>
**Pipes** chain commands using |.<br>
**Command History** navigate through previous commands(thanks to GNU Readline).<br>
**Robust Tokenization** handles quotes, spaces and special characters correctly.<br>

# Prerequisities
GNU Readline: Install the -devel libraries.<br>
Cmake.<br>
GCC.<br>

# Installation
Clone the repository:
```
git clone https://github.com/knight212121/shell.git
cd shell
```

Build and run the project:
```
./your_program.sh
```

# Limitations
This is not a complete bash replacement and you may encounter bugs. I'll continue working on it, but for basic use cases, it is ready!
